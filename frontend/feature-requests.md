# No assignee

Scenarios:

1. If a new task is created without an assignee then all the family members will be notified.
2. If the family member is removing the assignee from a task all the family members will be notified

Features:

- Task can be created without assignee
- If a new task is created without an assignee then all the family members will be notified.
- Anyone can remove the assignee from a task
- If the family member is removing the assignee from a task all the family members will be notified
